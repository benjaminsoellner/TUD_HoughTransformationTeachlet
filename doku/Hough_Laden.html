Bild laden

Die Basis für eine Hough-Transformation ist ein Bild. In diesem Schritt ist es möglich, ein Foto aus einer bestehenden Datenbank zu nutzen, oder alternativ ein eigenes hochzuladen. Im Schritt "Bild bearbeiten" werden im späteren Verlauf noch Werkzeuge zur Verfügung gestellt, mit denen das geladene Bild bearbeitet werden kann. So können zum Beispiel Linien hinzugefügt oder entfernt werden.
